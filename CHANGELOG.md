## [1.0.0] - 2024-02-15

### 新增
- ✨ 完整的 MCP 架构实现
- ✨ 7个专业 MCP 服务器
  - Sensor - 传感器数据管理
  - Weather - 天气服务
  - Crop Knowledge - 作物知识库
  - Decision - 决策引擎
  - Irrigation - 灌溉控制

- ✨ Web 界面（FastAPI + WebSocket）
- ✨ Arduino 硬件集成（ESP8266）
