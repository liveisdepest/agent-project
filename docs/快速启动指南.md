# 快速启动指南

## 🚀 5分钟快速开始

### 第一步：安装 uv 包管理器

```bash
# Windows (PowerShell - 以管理员身份运行)
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
```

### 第二步：安装并启动 Ollama

```bash
# 1. 下载 Ollama
# 访问 https://ollama.ai/download 下载安装

# 2. 拉取模型
ollama pull qwen2.5-coder

# 3. 验证安装
ollama list
```

### 第三步：配置环境变量

```bash
cd client/mcp-client

# 复制配置文件
copy .env.example .env

# 编辑 .env 文件，填入以下内容：
```

```env
API_KEY=ollama
BASE_URL=http://localhost:11434/v1
MODEL=qwen2.5-coder
OWM_API_KEY=你的天气API密钥
```

**获取天气 API Key**：
1. 访问 https://openweathermap.org/api
2. 注册免费账号
3. 复制 API Key

### 第四步：启动客户端

```bash
cd client/mcp-client
uv run client.py
```

### 第五步：测试系统

输入以下问题测试：

```
曲靖今天的天气怎么样，我的小麦需要灌溉吗？
```

## 🌐 启动 Web 界面（可选）

```bash
# 新开一个终端
cd server/web
uv run app.py

# 访问 http://localhost:8080
```

## 🔧 Arduino 硬件集成（可选）

### 1. 准备硬件
- ESP8266 开发板（NodeMCU）
- DHT11 温湿度传感器
- 土壤湿度传感器
- 继电器模块
- 水泵

### 2. 接线说明

```
DHT11:
  VCC  -> 3.3V
  GND  -> GND
  DATA -> D4 (GPIO2)

土壤湿度传感器:
  VCC  -> 3.3V
  GND  -> GND
  AO   -> A0

继电器:
  VCC  -> 5V
  GND  -> GND
  IN   -> D1 (GPIO5)

LED指示灯:
  正极 -> D0 (GPIO16)
  负极 -> GND (串联电阻)
```

### 3. 修改代码

编辑 `arduino_code.ino`：

```cpp
// 修改 WiFi 配置
const char* ssid = "你的WiFi名称";
const char* password = "你的WiFi密码";

// 修改服务器地址（改为你电脑的局域网IP）
const char* serverURL = "http://192.168.1.100:8080";
```

**查找电脑IP地址**：
```bash
# Windows
ipconfig

# 查找 "无线局域网适配器 WLAN" 下的 IPv4 地址
```

### 4. 上传代码

1. 使用 Arduino IDE 打开 `arduino_code.ino`
2. 选择开发板：工具 -> 开发板 -> ESP8266 -> NodeMCU 1.0
3. 选择端口：工具 -> 端口 -> COM?
4. 点击上传

### 5. 启动系统

```bash
# 1. 启动 Web 服务器
cd server/web
uv run app.py

# 2. Arduino 会自动连接并开始上报数据
# 3. 在 Web 界面查看实时数据
```

## 🔍 常见问题

### Q1: 提示 "未找到 API KEY"

**解决**：检查 `.env` 文件是否在正确位置（`client/mcp-client/.env`）

### Q2: MCP 服务连接失败

**解决**：
```bash
# 检查 Python 版本
python --version  # 需要 ≥3.10

# 重新安装依赖
cd client/mcp-client
uv sync
```

### Q3: 天气查询失败

**解决**：
1. 验证 `OWM_API_KEY` 是否正确
2. 检查网络连接
3. 确认 API Key 已激活（新注册需等待几分钟）

### Q4: Arduino 无法连接

**解决**：
1. 检查 WiFi 配置是否正确
2. 确认电脑和 Arduino 在同一局域网
3. 检查服务器地址是否正确（使用电脑的局域网IP）
4. 确认 Web 服务器已启动
